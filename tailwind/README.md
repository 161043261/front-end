# tailwind

## Project Setup

```sh
npm install
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
