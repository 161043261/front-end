# proj-demo

## Project Setup

```sh
npm install
npm install element-plus --save
npm install axios
npm install sass -D
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
